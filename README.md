# ScalaZen-IDE
## ScalaZen IDE, make visual WebGL Apps with Scala and ScalaZen Lib (Based on ThreeJs)

Install necesary dependencies with
```
./setup.sh
```

Run Application with
```
./run.sh
```

![Screenshoot](http://i.imgur.com/eepZySy.png?1)

#### Libraries used so far  
+ ScalaJS
+ ThreeJS
+ Bootstrap
+ NW.js
+ JQuery
